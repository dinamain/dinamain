Full-stack chat app with Django REST + WebSocket backend (Django Channels + Redis) and a React/Vite frontend. Features JWT auth, role-based group permissions, live typing indicators, presence tracking, message reactions, replies, and pinned messages. CI/CD via GitHub Actions, deployed on Render.
